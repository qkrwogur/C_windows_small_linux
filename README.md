# C_window_small_linux

윈도우 환경에서 사용할 수 있는 파일 문서 편집기 입니다.

명령어 종류
  cd 
  rmdir
  mkdir 
  rm
  vi
  pwd
  exit
  clear
  tree
  
  
vi 편집기는 미완성으로 오류가 많이 존재 합니다.
